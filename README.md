# Slay-the-Spire-2-Three-Kingdoms-Mod
This is a Three Kingdoms Kill mod for Slay the Spire 2 (currently in ongoing development; some bugs may occur, especially in online multiplayer mode, with continuous updates to follow)

这是一个杀戮尖塔2的三国杀mod(目前处于持续开发状态，可能会出现一些bug,尤其是联机模式，后续会不断更新)

New Additions: 1 new character, over 80 new playable cards (to be expanded to 90 in the future), 2 new playable relics (to be expanded to over 10 later). Feel free to send me a private message if you have any suggestions or ideas.

新增内容：1个新角色，80+新增可选卡牌（未来将拓展至90），2个新增可选遗物（未来拓展至10+），如果您有好的建议和想法，请私信我

**How to Install  如何安装**

Click the Release button on the right to download the zip file. Navigate to Manage > Browse Local Files in your Steam library. Open the mod folder in the file explorer (create one if it does not exist). Extract all contents of the downloaded zip file into this folder, and also extract the required dependency BaseLib into the same folder.

点击右侧release按钮下载压缩包，在steam界面找到管理-浏览本地文件，在文件管理器中打开mod文件夹（如果没有则新建），将压缩包内容解压到其中，同时将所需依赖BaseLib也解压到其中

BaseLib：https://github.com/Alchyr/BaseLib-StS2/releases/
